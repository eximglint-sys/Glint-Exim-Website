## Ruby Gemstone Image

**File Name:** `ruby.jpeg`

**Location:** Your ruby gemstone image is already in this folder.

**Current Image:** `ruby.jpeg`
- Recommended size: 1200x1200px or larger
- Format: JPG, PNG, or WebP
- The image should show the ruby gemstone with the golden Om symbol

**Where it's used:**
- Product selection page (`ruby-products.html`)
- Order summary in address/payment/success pages

If you have a different filename, update the `image` property in:
- `lp/ruby-products.js` (line 9)
